Описание программы решения уравнения теплопроводности
Выбор варианта запуска:
При запуске программы пользователю предлагается выбрать один из двух вариантов:

Запустить с изначальными параметрами:
Программа запускается с параметрами по умолчанию, которые включают:
Максимальное время моделирования: 5.0
Шаг времени: 0.01
Длина стержня: 1.0
Коэффициент теплопроводности: 0.001
Используется начальная функция температуры u(x, 0) = x * sin(5 * pi * x).

Изменить параметры:
Пользователь может ввести следующие параметры:
Максимальное время моделирования: Время, в течение которого программа будет моделировать распространение тепла.
Шаг времени: Интервал между каждым шагом моделирования.
Длина стержня: Длина материала, в котором моделируется теплопроводность.
Коэффициент теплопроводности: Показатель, определяющий скорость передачи тепла в материале.
Начальная функция температуры u(x, 0): Математическое выражение, определяющее начальное распределение температуры вдоль стержня.

Работа программы:
Программа вычисляет необходимое количество точек для численного решения уравнения теплопроводности в зависимости от заданных параметров. Затем:
Вычисляется шаг по пространству dx с учетом стабильности численного метода.
Заданные начальные условия (длина стержня, начальная функция температуры) используются для инициализации начального состояния стержня.
Выполняется анимация, показывающая эволюцию температуры вдоль стержня с течением времени.
По достижении максимального времени моделирования программа останавливается. Анимация автоматически приостанавливается, и график остается статичным.
Визуализация результатов:
Во время анимации на экране отображается график, показывающий распределение температуры в стержне в разные моменты времени. При достижении максимального времени моделирования анимация автоматически приостанавливается
